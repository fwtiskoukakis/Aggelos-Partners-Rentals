# Aggelos Rentals Partners

Partner portal for hotel guests (The Rooster Antiparos). Fleet browser and WhatsApp request flow.

- **Stack:** Next.js, TypeScript, Tailwind CSS
- **Deploy:** Vercel (see [DEPLOY.md](./DEPLOY.md))

## Push to GitHub (first time)

1. Create an empty repo on GitHub named **aggelos-rentals-partners** (or **aggelos-rentals-partners**).
2. Run (replace `YOUR_USERNAME` with your GitHub username):

```bash
cd "c:\Users\Fotis\Desktop\Aggelos Rentals Piraeus Website\antiparosrentacar.com\aggelos-partners-rentals"
git remote add origin https://github.com/YOUR_USERNAME/aggelos-rentals-partners.git
git push -u origin main
```

3. In [Vercel](https://vercel.com), import this repo and deploy.
